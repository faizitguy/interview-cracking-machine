How to read it (top → bottom = a request; the sides = the response): You act in the UI → the Backend spawns the headless Claude Code engine on your Max plan (no API key) → Claude reads & edits the plain data files, which are the single source of truth. Coming back up: Claude streams its progress into the assistant panel (violet, left), and the backend's file watcher detects the changed files and pushes a live update to the UI over WebSocket (teal, right) — so the dashboard re-renders by itself. That closed loop is the whole app.
